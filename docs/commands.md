# Документация
Есть возможность изменения draw.io схемы с помощью ai агента.

В моём случае я использую Qwen

## Промт
```
Задача: [кратко что нужно изменить]

Файлы:
1. Payship_Core_architecture.drawio.xml — текущая схема
2. project_docs.txt — документация пакетов (go doc -all ./...)
3. tree.txt — структура каталогов
4. [опционально] содержимое конкретных файлов, которые меняются

Контекст:
- [что добавилось/удалилось/изменилось в коде]
- [какие новые связи нужно показать]
```

## Генерация
```
Генерируюмые файлы для контекста
```bash
bash docs/generate_docs.sh
# Вывод в ./context, при запуске из корня
```
